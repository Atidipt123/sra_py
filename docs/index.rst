sra\_py
=======

A python wrapper for `some-random-api <https://some-random-api.ml>`__

Requirements
------------

`Python 3.8.6 or more <https://python.org>`__

Installation
------------

.. code:: cmd

    pip install sra_py

Example
-------

.. code:: py

    # Random Dog Image

    import sra_py as sra

    dog = sra.dog()

    print(dog)
